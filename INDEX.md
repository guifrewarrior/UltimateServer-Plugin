## 🚀 Getting Started in 3 Steps

1. **Download** → Get JAR from releases
2. **Install** → Copy to `plugins/` folder
3. **Configure** → Add Supabase credentials to `config.yml`
4. **Restart** → Server will start with plugin

See [QUICK_START.md](QUICK_START.md) for details.
